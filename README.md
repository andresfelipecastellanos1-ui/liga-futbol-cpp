# Sistema de Liga de Fútbol

Proyecto en C++ que permite gestionar una liga de fútbol desde consola.

## Funcionalidades

- Registrar partidos
- Generar tabla de posiciones
- Diferencia de gol (DG)
- Validación de partidos repetidos
- Control de equipos por fecha
- Guardado en archivos (.txt)
- Reinicio de liga

## Archivos

- config.txt → configuración
- partidos.txt → partidos
- fechas.txt → historial
- tabla.txt → tabla generada

## Ejecución

```bash
g++ main.cpp -o liga
./liga
